## 🚀项目指令

### 基本指令

- 启动前请先安装依赖`pnpm install`

```bash
# 启动demo项目
pnpm run dev:dev

# 启动文档
pnpm run docs:dev

# 组件库打包
pnpm run build:lib

# 文档打包
pnpm run docs:build
```

### 新建组件

- 通过命令行交互快速生成组件模板

```bash
pnpm run gen:component
# Q1:组件名称
# Q2:组件描述(中文名称)
# Q3:选用模板(选.tsx)
```
